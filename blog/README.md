# FindNeedle Blog

A modern blog for the FindNeedle project, powered by VitePress.

## Quick Start

```bash
cd blog
npm install
npm run dev
```

## Commands

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Structure

```
blog/
├── docs/              # Markdown content
├── .vitepress/        # VitePress config
├── package.json
└── README.md
```
